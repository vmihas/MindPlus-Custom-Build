# MindPlus-Custom-Build
A Custom Build of Mind Plus for Primary Education
### ONLINE & OFFLINE mode
Υποστηρίζονται όλοι οι αισθητήρες/ενεργοποιητές του κιτ.
Επιπλέον υποστηρίζεται και ο πομπός και ο δέκτης υπερύθρων. Μπορείτε να χρησιμοποιήσετε το τηλεκοντρόλ που υπάρχει στο R3 κιτ.

### Βελτιώσεις
Γίνεται χρήση όμορφων UI widgets όπως:

> Πιανάκι για τις νότες στο buzzer

> Γραφικό τηλεχειριστήριο (customized ειδικά για το δικό μας κιτ) για τα blocks υπερύθρων

> Παλέτα για την επιλογή χρώματος των rgb neopixel

* Για τα neopixel χρησιμοποιείται η ενσωματωμένη βιβλιοθήκη της DF_Robot.
* O κινητήρας μπορεί πλέον να συνδεθεί και στο SPI port.
* Τα blocks είναι οργανωμένα σε κατηγορίες για εύκολη εύρεση.
* Απενεργοποιήθηκε ο έλεγχος ενημερώσεων και το αντίστοιχο popup window.
* Σε νέα εγκατάσταση η προεπιλεγμένη γλώσσα είναι τα ελληνικά.
* Διάφορες άλλες βελτιώσεις "εδώ και εκεί"..

### Installation
Έχω υλοποιήσει δύο installers:
* Mind+ Custom Build Full Installer
* Mind+ Custom Build Update Installer

Όταν κάποιος εγκαθιστά το Custom Build για πρώτη φορά, θα πρέπει να τρέξει το **_Mind+ Custom Build Full Installer_** (περιέχει και το official Mind+ bundle που απαιτείται),
ενώ τις υπόλοιπες φορές το **_Mind+ Custom Build Update Installer_** (περιέχει udpates/προσθήκες των πρόσθετων) που είναι πολύ μικρότερο σε μέγεθος και πιο γρήγορο στην εγκατάσταση.
